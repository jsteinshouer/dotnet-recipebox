
# RecipeBox

A simple recipe application using .net core.

## Requirements

- .NET CLI v8
- .NET SDK v8
- Node.js v18+

You can also use [Github Codespaces](https://codespaces.new/jsteinshouer/dotnet-recipebox).

## Getting Started

### Backend

```
cd backend
dotnet run
```

### Front-end

```
cd ui
npm install
npm run start
```
